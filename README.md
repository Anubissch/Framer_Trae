# Framer Trae Components

## What these files are:

### Qsd.tsx
This is a **custom Framer component**!
- Creates an animated rounded square
- Has a color property (tint) you can change in Framer
- Scales up and rotates on hover
- Uses Framer Motion for animations!

### Examples.tsx
These are **Framer "Overrides"**!
These are higher-order components you can apply to other Framer elements:
- `withRotate`: Makes elements rotate 90 degrees over 2 seconds
- `withHover`: Makes elements scale up slightly on mouseover
- `withRandomColor`: Changes the element's background color randomly when clicked!

## How to use:
1. Open Framer
2. Create a new code component or override
3. Copy & paste these files into your Framer project
4. Use them like any other Framer component!
